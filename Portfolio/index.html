SKILLS & ABILITIES

HTML

HTML (Hypertext Markup Language) is used for creating the structure and content of web pages. It provides a way to mark up text and add images, videos, and links. HTML is not a programming language but rather a markup language.

CSS

CSS (Cascading Style Sheets) is used to style HTML elements and make web pages look visually appealing. It provides a way to control the layout, typography, color, and other aspects of the presentation of a web page.

PYTHON

Python is a high-level programming language that is widely used for data analysis, scientific computing, web development, and machine learning. It has a large and active community of developers and a vast ecosystem of libraries and tools that make it a versatile language for a wide range of applications.

SQL

SQL (Structured Query Language) is used to manage and manipulate data in relational databases. It allows users to retrieve, insert, update, and delete data from databases

JAVA SCRIPT

JavaScript is a programming language used to create interactive elements on web pages. It can be used for form validation, animations, and other dynamic features on a website.

C LANGUAGE

C is a general-purpose programming language that is extremely popular, simple, and flexible to use. It is a structured programming language that is machine-independent and extensively used to write various applications, Operating Systems like Windows, and many other complex programs.
